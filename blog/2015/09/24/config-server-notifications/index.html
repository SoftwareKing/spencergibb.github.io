config server notifications

Sep 24, 2015 · 1 minute read

This commit (8d8e07a78) in spring-cloud-config added support for push notifications to config clients.

<dependency>
  <groupId>org.springframework.cloud</groupId>
  <artifactId>spring-cloud-config-monitor</artifactId>
</dependency>
<dependency>
  <groupId>org.springframework.cloud</groupId>
  <artifactId>spring-cloud-starter-bus-amqp</artifactId>
</dependency>

Manual push

http --form POST :8888/monitor path=configserver
